# Autonomous Scientific AI Experimenter

This example demonstrates a simplified 'autonomous scientific AI' agent. It simulates an AI iteratively designing, conducting, and analyzing experiments to discover optimal conditions for a black-box scientific process. This mimics how AI could evolve beyond chatbots to become an active partner in scientific discovery, autonomously driving research.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`
